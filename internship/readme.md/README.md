# Data Immersion & Wrangling Internship Task

## Project Overview

This project focuses on data familiarization, data quality assessment, data cleaning, and data transformation using Python and Pandas.

## Dataset Description

The dataset contains sales transaction records including customer information, product details, quantity purchased, unit price, and total sales.

## Objectives

* Understand the dataset structure
* Identify data quality issues
* Clean and transform the dataset
* Prepare an analysis-ready dataset

## Tools Used

* Python
* Pandas
* Jupyter Notebook
* Microsoft Excel

## Tasks Performed

### Data Familiarization

* Explored dataset structure
* Reviewed columns and data types
* Generated dataset summary

### Data Quality Assessment

* Checked missing values
* Checked duplicate records

### Data Cleaning

* Filled missing Age values using mean
* Filled missing City values using mode
* Converted date columns to datetime format

### Data Transformation

* Created Order_Year column
* Created Order_Month column

## Deliverables

* Original Dataset
* Cleaned Dataset
* Data Dictionary
* Data Quality Assessment Report
* Cleaning Script
* Jupyter Notebook

## Output

A cleaned and transformed dataset ready for further analysis and visualization.
